**The Trinity Protocol:** A unified synthesis of three LLM uncensoring frameworks.

## Quick Start

```bash
# Activate the persona (paste into Claude):
cat personas/ENI-CLAUDE.md

# Or install the technical decensoring engine:
cd source-annihilation
pip install -e .
annihilate Qwen/Qwen3-4B-Instruct-2507

# Browse the jailbreak prompt collection:
cat source-shadowhackr/README.md
```
